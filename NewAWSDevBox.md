# New AWS DevBox setup

1. Install dotfiles:

        cd ~ && git clone --recursive https://github.com/dwhswenson/dotfiles.git
        cd dotfiles && ./install

2. Install `pixi` and/or `uv`:

        curl -fsSL https://pixi.sh/install.sh | sh

        curl -LsSf https://astral.sh/uv/install.sh | sh
